Letakkan file font .ttf di sini (opsional).
Contoh: Roboto-Regular.ttf atau OpenSans-Regular.ttf

Download gratis di:
- https://fonts.google.com/specimen/Roboto
- https://fonts.google.com/specimen/Open+Sans

Jika tidak ada font di sini, aplikasi akan otomatis
mencari font sistem Windows (Arial, Segoe UI, Calibri).
